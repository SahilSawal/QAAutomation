# QAAutomation
Design QA automation tool. which can work on all web applications.
Goals :
Spring AI test case extraction
Selenium MCP/ playwright MCP agent execution loop
HTML pass/fail report generation

OPTIONAL : Can use OpenAI/ Anthropic.
Phase 1: 
Convert the user guide to test cases using Spring AI.
Phase 2: 
execute the test cases using MCP.
Phase 3:
generate the report;
phase 4 : optional 
maintain the log and screenshot of failed and passed cases
